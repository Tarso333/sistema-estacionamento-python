# Sistema de Estacionamento 🚗

Projeto desenvolvido em Python com MySQL para controle de vagas de estacionamento.

## Funcionalidades

- Cadastro de veículos
- Registro de entrada
- Registro de saída
- Controle de vagas disponíveis
- Vagas prioritárias
- Listagem de veículos estacionados
- Relatório de veículos do dia

## Tecnologias

- Python
- MySQL
- VS Code 

## Como rodar o projeto

1. Instale MySQL
2. Execute o arquivo `database.sql`
3. Configure as credenciais no `main.py`
4. Rode:

```bash
python main.py
